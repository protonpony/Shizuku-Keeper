# Shizuku Keeper
Powerful Automate flows for Android 11+ that start Shizuku on boot and keep it running perpetually via ADB.

### [Shizuku Keeper ♾️](https://llamalab.com/automate/community/flows/51118)  
Continuously run Shizuku and ADB uninterrupted without root or Wi-Fi using ADB in TCP/IP mode and Automate Premium.

### [Shizuku Keeper ♾️ Lite](https://llamalab.com/automate/community/flows/51012)  
Check Shizuku at regular intervals and automatically restart it using wireless debugging with the free version of Automate.
